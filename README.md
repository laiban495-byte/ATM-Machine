## 📌 About The Project
This project is a modern, object-oriented ATM Simulator built using Python. It features a premium, interactive Graphical User Interface (GUI) modeled after the HDFC Bank ATM experience, utilizing the `customtkinter` library for a sleek, responsive design.

### 🌟 Key Features
- **Premium UI/UX:** Clean, modern interface with corporate branding (HDFC Blue & Red).
- **Physical ATM Feel:** Fully functional digital keypad mapped to ATM operations.
- **Core Banking Logic:** Securely handles deposits, withdrawals, and balance inquiries.
- **Transaction History:** Real-time tracking and viewing of recent transactions (Mini Statement).
- **Modular Architecture:** Clean separation of frontend (GUI) and backend (Core Logic) via Python packages.

## 💻 Tech Stack
* **Language:** Python 3.x
* **GUI Framework:** CustomTkinter (Modern extension of Tkinter)

## 🚀 Getting Started

### Prerequisites
Make sure you have Python installed. Then install the required dependencies:
```bash
pip install -r requirements.txt
```

### Running the Application
To launch the ATM Simulator with the modern GUI:
```bash
python gui_main.py
```
To run the CLI (Command Line Interface) version:
```bash
python main.py
```
